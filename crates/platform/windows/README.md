# Flor Platform Windows

Windows 平台支持 crate，提供窗口创建、消息循环、输入处理等平台相关功能。

## Features

| Feature        | 说明            |
|----------------|---------------|
| `theme-change` | 系统主题变化监听      |
| `monitor`      | 多显示器信息获取      |
| `drag-drop`    | OLE 拖放支持      |
| `tray`         | 系统托盘图标支持      |
| `hi-dpi`       | 高 DPI 感知      |
| `win7-compat`  | Windows 7 兼容性 |

## 功能

- 窗口创建与管理
- Win32 消息循环
- 键盘/鼠标输入处理
- IME 输入法支持
- 鼠标捕获
- 窗口操作 (最大化/最小化/关闭等)
